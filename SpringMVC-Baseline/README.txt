Introduction:
Spring MVC based web application which I'm using for my blog posts about Spring MVC.
  
Prerequisites:
1. JDK 1.7 or above
2. Apache Maven 3.0.1 or above

Starting Web server:
  mvn jetty:run-exploded
  
Access using the browser:
 http://localhost:8080/spring-mvc-baseline/do/customers/list
